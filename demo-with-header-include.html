Trang Demo với Header Include

Đây là trang demo sử dụng header được tách riêng thành file include.

Cách sử dụng Header Include:

  1. Thêm placeholder div: <div id="header-placeholder"></div>
  2. Thêm placeholder cho mobile menu: <div id="mobile-menu-placeholder"></div>
  3. Thêm placeholder cho cart sidebar: <div id="cart-sidebar-placeholder"></div>
  4. Thêm script include: <script src="assets/js/includes.js"></script>

Lợi ích:

  • ✅ Header nhất quán trên tất cả các trang
  • ✅ Dễ dàng cập nhật header chỉ bằng cách sửa 1 file
  • ✅ Giảm code trùng lặp
  • ✅ Dễ bảo trì và quản lý
